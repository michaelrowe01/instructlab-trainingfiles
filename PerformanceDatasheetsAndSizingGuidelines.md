## CLM 5.x

### CLM 5.0.2

For the 5.0.2 release, performance testing validated that there were no
regressions for RTC, RQM and RELM between the 5.0, 5.0.1 and 5.0.2
releases, therefore there has not been a need to create 5.0.2 datasheets
for those products. For 5.0.2 performance information for RTC and RQM
please consult the 5.0 datasheets. For the 5.0.2 performance information
for RELM please consult the 5.0.1 datasheet.
